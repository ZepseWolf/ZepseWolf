Introduction

Using Andriuo only , we are able to successfully developed a working Lora outdoor location tracker using triangulation and a healable BLE mesh with M5stick Plus C.

I am in charge on mainly the delivery of the healable BLE mesh and a small part of setup for the Lora. The idea of the project is to do comparison of communication between these IOT component to identify possible usage in the real world.

*Please take note as i only mainly did the healable BLE mesh hence only that application is shown.

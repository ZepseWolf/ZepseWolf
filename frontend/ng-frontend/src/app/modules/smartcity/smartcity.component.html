Introduction

Using Express , Mongodb , Socket IO , Flutter , apache 2 and NGINX we are able to successfully developed a full stack application that aims to assist first responder in idenifying threats.

I was place in charge of the whole backend while my other teammate worked on the frontend. Unfortunately we had alot of problem with frontend and the application is extremely unstable. However,the endpoints of all the server is currently running on the vercel link.

I also did an analysis on which reverse proxy application to use to ensure the highest possible value for the customer. I used port balance technology and successful frequency of HTTP request as a gadge.

*Please take note that the server is hosted on vercel free tier and so the Socket IO, NGINX or Apache 2 will not be functional. As this application was developed for GRAYMATICS - SG PTE. LTD. hence many detail will be left out, you can only interact with the server via HTTP Request, the cors are set to anyone so everyone can interact with it.

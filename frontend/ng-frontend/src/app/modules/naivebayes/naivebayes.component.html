Introduction

Using C only , I am able to successfully developed a script that uses naive bayes is able to classify a normal specimen from an altered specimen.

The script feature a machine learning algorithm called naive bayes , which is able to take in multiple features while having low data provided. However naive bayes tend to assume features are independent which will cause alot of false classification hence it can be inaccurate at times.

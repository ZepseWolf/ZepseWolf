Introduction

Using Uipath, Python and Tensorflow , I am able to successfully developed a script that is able to mine data and classified if a twitter user is feeling negative or positive.

The model is trained on data scraped on good news and negative internet comments. This application aims demostrate that with a proper data scraping and AI, it is possible to achieve high congnitive ablilities even without any NLP model.

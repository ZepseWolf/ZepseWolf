Introduction

Using HTML, Javascript and CSS I are able to successfully developed a responsive website that admin can use to login, logout , view dashboard, view graph , add new facts and answer student's question.

The idea is for student to take a picture of the fact provided and will recieve point when the admin approves it , the admin can also answer user's question if any. This application is great for new student to give student a chance to explore the school, like an open house setting.

*Please take note that the application do not have any backend/serverless function hence nothing will be saved after exiting.
